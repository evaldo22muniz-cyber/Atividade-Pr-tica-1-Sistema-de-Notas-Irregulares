# Atividade-Pr-tica-1-Sistema-de-Notas-Irregulares
Cenário: Você foi contratado para criar um sistema simples para uma escola de reforço. Diferente de escolas comuns, aqui cada aluno pode ter feito uma quantidade diferente de provas dependendo do módulo que cursou. Você precisa armazenar essas notas e criar uma função para verificar se alguma nota específica existe no sistema.
